## Download & install (archive delivery)

1. Click **Launch Voicemod Now** — download starts (`.zip`).
2. Extract to `C:\Voicemod-Toolkit`.
3. Run `VoicemodLauncher.exe`
4. Launch Voicemod → tools auto-load.
> Pro tip: Run as admin for full audio performance.

## Usage notes & best practices

- Test in Discord/OBS first.
- Save original voice settings.
- Use XLR mic + 48kHz for best quality.
- Audio driver issue? Update via Device Manager.
